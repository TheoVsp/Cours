Les 2 types de balises

Il existe deux types de balises :

  1. les balises usuelles encadrent du code par exelple : <body> et </body>
  2. les balises dites orphelines n'encadrent aucun code ; elles se ferment elles-mêmes. Par exemple <br/> abréviation de break row; <hr/> abréviation de horizontal row;
